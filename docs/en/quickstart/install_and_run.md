# Install and Run

```bash
sudo dpkg -i fourier-grx-x.x.x.deb
fourier-grx install
fourier-grx start
```

The installed launcher expands the packaged ZIP into `$HOME/fourier-grx`, runs setup scripts, lets you choose the HXC variant, and then starts `run.bin` with the selected config file.
