## Stable Releases

The repository currently publishes two kinds of stable outputs:

1. GitHub Pages documentation via `.github/workflows/pages.yml`
2. Runtime build artifacts from the root `Makefile`

## Preview Releases

No separate preview release list is maintained at this time.

## Installation

For the runtime installation flow, refer to [Install and Run]({{ '/docs/en/quickstart/install_and_run/' | relative_url }}).

## Release Notes
