# Usage

The installed command set is implemented by `release/deb_run.sh`.

Common commands:

- `fourier-grx config`
- `fourier-grx list`
- `fourier-grx start`
- `fourier-grx stop`
- `fourier-grx background`
- `fourier-grx enable_service`
- `fourier-grx disable_service`
- `fourier-grx setup_conda`
- `fourier-grx setup_conda_py310`
